As far as setting up goes

I'm assuming Git is already downloaded and usable from the command line.
Let's work on some setting configurations. Git needs to know your name right? Open up that command line.
git config --global user.name "My Name"
git config --global user.email myemail@provider.com

Navigate to the repository you want version control on -- Git, let's git init a repo locally. So now we've made
your local repo a Git repo. Meanwhile, find your friend GitHub and make a repository with the same name on there.
If your repository is empty,

  • Simply create a file and add to your repo (like a README). We'll cover commit & pushing later.
Otherwise,
  • Add a remote using
    git remote add origin https://github.com/yourname/yourreponame.git
    git push -u origin master

You can check the status of the modifications you've made to your repo using
git status

Get ready to commit
